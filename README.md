# pyQuickOllama

这是一个能够快速获取 `ollama` 模型列表并打开模型的命令行的工具  
该工具可以避免频繁的输入 `ollama run xxxx` 和 `ollama list`  
提高日常使用的流畅度与舒适度

# 说明


**当然需要先安装ollama**

**目前不支持连续对话与其他高级功能**

重构之后不使用命令而是使用 **网络请求** 来获取获取回答  
可以避免原版程序在终端下中文删除的 BUG

首次启动会生成 `config.ini` ，文件说明如下

```ini
[DEFAULT]
// 如不指定，会出现选择菜单
// 如果指定，启动会跳过选择菜单直接进入对话
model =
// 是否开启打字机输出
stream = True
```

模型列表默认会按照 `family` 重新排序,而不是默认顺序
